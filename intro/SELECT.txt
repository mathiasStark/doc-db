PROJEÇÃO
	UTILIZADO PARA PROJETAR CONSTRUIR DADOS QUE EXISTEM OU NÃO EM UMA TABELA.
	UTILIZAR O COMANDO SELECT SEGUIDO DO NOME DA COLUNA(S) QUE DEJEJA EXIBIR OS VALORES + FROM NOME DA TABELA(S) PARA PROJETAR DADOS DA TABELA NA TELA;
	
	#RETORNA DADOS DE UMA TABELA
	SELECT 'NOME DA COLUNA' FROM 'NOME DA TABELA';

	#RETORNA HORA ATUAL DO SISTEMA
	SELECT NOW() AS DATA_HORA;
	
	NÃO UTILIZAR A PROJEÇÃO COM * PARA PROJETAR TODAS AS COLUNAS E OS DADOS DO BANCO DE DADOS, PODE OCASIONAR NO CRACH DO BANCO DEVIDO A GRANDE QUANTIDADE DE DADOS, OU TRAZER DADOS DESNECESSARIOS OCUPANDO ESPAÇOS NA REDE CAUSANDO UM GARGALO  NO SISTEMA TRAFEGANDO DADOS A MAIS;
