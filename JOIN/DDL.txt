DDL - DATA DEFINITION LANGUAGE

UTILIZADO PARA DEFINIR A ESTRUTURA DA TABELA, COMO NOME DAS COLUNAS, TIPO DOS DADOS DA COLUNAS.

COMANDOS:
ALTER TABLE, CREATE TABLE, DROP TABLE.

ALTERANDO NOME E TIPOS DA COLUNA UTILIZANDO CHANGE:
ALTER TABLE <NOME TABELA> CHANGE <NOME COLUNA> <NOVO NOME COLUNA> <TIPO>;

ALTERANDO APENAS O TIPO UTILIZANDO MODIFY:
ALTER TABLE <NOME TABELA> MODIFY <NOME COLUNA> <NOVO TIPO>;

ADICIONANDO NOVAS COLUNAS COM ADD:
ALTER TABLE <NOME TABELA> ADD <NOME DA COLUNA> <TIPO>;

APAGANDO UMA COLUNA COM DROP COLUMN;
ALTER TABLE <nome tabela> DROP <NOME COLUNA>;

ADICIONANDO COLUNA NA ORDEM ESPECIFICA COM ADD COLUMN AFTER E FIRST:
ALTER TABLE <NOME TABLE> ADD COLUMN <NOME COLUNA> <TIPO> AFTER OU FIRST <NOME COLUNA>:
